# RSNA-Pediatric-Bone-Age-Challenge-2017-
# Dataset
Questo progetto utilizza il dataset **RSNA Pediatric Bone Age Challenge 2017** per l'addestramento e la validazione del modello di stima dell'età ossea.

**I file del dataset non sono inclusi nel repository** a causa delle dimensioni e delle condizioni di utilizzo imposte da RSNA.

Prima di eseguire il codice è necessario scaricare manualmente:

* Training Dataset (~9.6 GB)
* Training Dataset Annotations
* Validation Dataset (~1.1 GB)

I file sono disponibili nella pagina ufficiale della challenge RSNA:

https://www.rsna.org/artificial-intelligence/ai-image-challenge/rsna-pediatric-bone-age-challenge-2017

è sufficiente scaricare gli archivi nella cartella principale del progetto.

## Informazioni sul dataset

Il dataset della challenge RSNA 2017 contiene radiografie della mano sinistra di pazienti pediatrici annotate con l'età ossea. La versione originale della challenge comprende:

* 12.611 immagini per il training
* 1.425 immagini per la validazione
* 200 immagini per il test finale della competizione

Il dataset è distribuito esclusivamente per scopi di ricerca, istruzione e altri usi non commerciali.
